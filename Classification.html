Public

Meaning
Content that can be shared with anyone
Access
Full access (view, edit, share)
Users
Internal + External
Examples
Brochures, website content
Ensure no sensitive data is included

Internal Restricted

Meaning
Internal data for Merino employees only
Access
Full editing allowed
Users
@merinoservices.com only
Examples
SOPs, internal files
Do not share outside the organization

Confidential

Meaning
Sensitive internal data
Access
View only, reply allowed
Users
@merinoservices.com only
Examples
Financial reports, internal docs
Use External labels when sharing with clients

Highly Confidential

Meaning
Highly sensitive data
Access
View only, restricted
Users
Authorized users only
Examples
Legal, salary data
Share only with approved users

External – Restricted

Meaning
External collaboration documents
Access
View and edit allowed
Use only when external editing is required

External – Confidential

Meaning
Client sharing documents
Access
View only
Verify recipient before sharing

External – Highly Confidential

Meaning
Highly sensitive external data
Access
Strict view only
Do not share beyond intended recipients
